Theming & Internationalization Demo

Demonstrates CSS custom properties for theming and i18n for validation messages.

🎨 CSS Custom Properties Theming

The component exposes CSS custom properties for easy theming:
--ck-error-color, --ck-border-radius, --ck-focus-color, --ck-disabled-opacity 
/* Default theme (no overrides) */
ck-editable-array {
  --ck-error-color: #dc3545;
  --ck-border-radius: 4px;
  --ck-focus-color: #0066cc;
  --ck-disabled-opacity: 0.5;
}

🌐 Internationalization (i18n)

Use the i18n property to customize validation error messages. Click "Edit" and clear fields to see localized error messages. 
// English (default)
el.i18n = {
  required: (field) => `${field} is required`,
  minLength: (field, min) => `${field} must be at least ${min} characters`
};
