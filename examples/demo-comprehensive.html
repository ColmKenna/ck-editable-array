🔗 Feature 3: Nested Property Binding

Core

Try this: Edit a row to see nested property paths in action. The component supports dot notation like data-bind="person.name" and data-bind="address.city" for complex data structures.

🔒 Feature 4: Exclusive Locking

Core

Try this: Click "Edit" on any row. Notice that other rows become locked (grayed out and disabled) and the Add button is disabled. Only one row can be edited at a time, preventing data conflicts.
