ck-editable-array – Modal Edit with Validation

This example demonstrates validation in modal edit mode. The modal form shows:

  • Required field validation – Name and email must be provided
  • Field-level errors – Error messages appear under each field using [data-field-error]
  • Row summary – [data-error-summary] displays a count of validation errors
  • Automatic Save disable – Save button is disabled until all validation rules pass
  • Visual feedback – Invalid inputs get aria-invalid="true" and red styling
