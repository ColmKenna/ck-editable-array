CSS snippet: set variables on the host

ck-editable-array {
  --cea-background: #ffffff;
  --cea-padding: 16px;
  --cea-border-radius: 12px;
  --cea-message-color: #0b5ed7;
  --cea-focus-color: #dc3545;
  --cea-focus-background: rgba(220, 53, 69, 0.12);
}

CSS part: ::part(rows)

The rows container is exposed as a part named rows and can be styled from outside the component.

ck-editable-array::part(rows) {
  border: 1px dashed #999;
  padding: 12px;
  border-radius: 12px;
}
