Button Customization

Demonstrates button text customization and CSS parts styling for themeable action buttons.

1. Default Buttons (Baseline)

No customization applied - shows default English button text.

<ck-editable-array>
  <!-- No button attributes - uses defaults: Edit, Save, Cancel, Delete -->
</ck-editable-array>

3. Translated Buttons (i18n)

Support internationalization by changing button text to any language.

<!-- French example -->
<ck-editable-array
  button-edit-text="Modifier"
  button-save-text="Enregistrer"
  button-cancel-text="Annuler"
  button-delete-text="Supprimer"
  button-restore-text="Restaurer">
</ck-editable-array>

4. Icon-Only Buttons

Set text to empty string and use CSS background images for compact, icon-only buttons.

<ck-editable-array
  button-edit-text=""
  button-save-text=""
  button-cancel-text=""
  button-delete-text=""
  button-restore-text="">
</ck-editable-array>

<style>
ck-editable-array.icon-only::part(button) {
  width: 32px;
  height: 32px;
  background-size: 16px 16px;
  background-position: center;
  background-repeat: no-repeat;
  text-indent: -9999px; /* Hide empty text */
}

ck-editable-array.icon-only::part(button-edit) {
  background-color: #3b82f6;
  background-image: url('data:image/svg+xml...');
}
/* ... similar for other buttons ... */
</style>

5. Gradient Styled Buttons

Apply custom colors, gradients, and hover effects using CSS parts.

<ck-editable-array
  button-edit-text="Edit Row"
  button-save-text="Save Changes">
</ck-editable-array>

<style>
ck-editable-array.styled-buttons::part(button-edit) {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  border: none;
  padding: 8px 16px;
  border-radius: 6px;
}

ck-editable-array.styled-buttons::part(button-edit):hover {
  transform: translateY(-1px);
  box-shadow: 0 4px 8px rgba(102, 126, 234, 0.4);
}
</style>

6. Minimal Buttons

Clean, minimal button design with subtle hover states.

<ck-editable-array
  button-edit-text="edit"
  button-save-text="save">
</ck-editable-array>

<style>
ck-editable-array.minimal-buttons::part(button) {
  border: 1px solid #e5e7eb;
  background: white;
  padding: 4px 12px;
  border-radius: 4px;
  font-size: 0.875rem;
  transition: all 0.15s;
}

ck-editable-array.minimal-buttons::part(button-edit):hover {
  border-color: #3b82f6;
  color: #3b82f6;
}
</style>

Summary

Button customization provides:

  • Text Customization: 5 attributes (button-edit-text, button-save-text, button-cancel-text, button-delete-text, button-restore-text)
  • CSS Parts: ::part(button), ::part(button-edit), ::part(button-save), ::part(button-cancel), ::part(button-delete)
  • Use Cases: Emojis, internationalization, icons, custom themes, minimal designs
  • Backward Compatible: All defaults match original behavior (Edit, Save, Cancel, Delete, Restore)
