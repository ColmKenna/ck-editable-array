ck-editable-array – CSS Custom Properties (Theming)

This example demonstrates how to use CSS custom properties to theme the component. The component exposes several --ck-* variables that can be overridden on the component element or any ancestor.

Select a Theme

Live Demo

Try entering invalid data (empty or too short) to see themed validation styles.

CSS Custom Properties

The following CSS variables are available for theming:

Property Default
--ck-row-padding 12px
--ck-error-color #dc3545
--ck-border-radius 4px
--ck-border-color #ddd
--ck-focus-color #0066cc
--ck-disabled-opacity 0.5
